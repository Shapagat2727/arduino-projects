# LED Brightness Control with Potentiometer

## Overview
This project reads analog voltage from a potentiometer and uses it to control the brightness of an LED using PWM.

## Components Used
- Arduino board
- Potentiometer
- Red LED
- 1kΩ resistor
- Jumper wires
- Breadboard

## How It Works
The potentiometer changes the input voltage.  
The Arduino reads this analog voltage value and maps it to a PWM output value to control the LED brightness.

Turning the potentiometer:
- increases brightness
- decreases brightness
